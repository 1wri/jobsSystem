# python招聘推荐系统带万字论文和ppt

基于python的招聘推荐系统 Python 从某招聘网站抓取海量招聘数据， 进行数据清洗和格式化后存储到关系型数据库中（如mysql），利用 Django + MySQL + Echarts 搭建招聘信息可视化分析系统，实现不同岗位的学历要求、工作经验、技能要求、薪资待遇等维度的可视化分析。同时依据用户需求，运用协同过滤推荐算法来实现热门岗位的推荐。
